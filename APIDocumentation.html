get /query
Search documents sorted by BM25 score (queryDocuments)

Query parameters

query (required)
Query Parameter — The search query to find matching documents
limit (optional)
Query Parameter — The number of documents to return per page default: 100
page (optional)
Query Parameter — The page number to retrieve default: 1
max_summary_len (optional)
Query Parameter — The maximum length of the document summary to return default: 4000

Return type

array[Document]

Example data

Content-Type: application/json
[ {
  "SOURCE_ID" : 0,
  "SUMMARY" : "SUMMARY",
  "DOCUMENT_URL" : "DOCUMENT_URL",
  "TITLE" : "TITLE",
  "PUBLISH_DATE" : "2000-01-23",
  "DOCUMENT_TYPE" : "DOCUMENT_TYPE",
  "DOCUMENT_LANGUAGE" : "DOCUMENT_LANGUAGE"
}, {
  "SOURCE_ID" : 0,
  "SUMMARY" : "SUMMARY",
  "DOCUMENT_URL" : "DOCUMENT_URL",
  "TITLE" : "TITLE",
  "PUBLISH_DATE" : "2000-01-23",
  "DOCUMENT_TYPE" : "DOCUMENT_TYPE",
  "DOCUMENT_LANGUAGE" : "DOCUMENT_LANGUAGE"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully retrieved the ranked documents
